企鹅工坊 v1.5.1 更新日志
========================

发布日期：2026年1月19日

【RunningHub 应用配置节点重大升级】

✨ 新功能
--------
1. 批次执行支持
   - rh-config 节点现在支持批次选择（1-9次）
   - 选择批次数后会自动发送多次独立任务
   - 每个任务生成独立的输出节点，自动向下排列

2. 统一的浮动控制条
   - rh-config 节点的控制条移至节点上方
   - 与图片节点等保持一致的交互体验
   - 包含：批次选择器 (- N +)、RUN、下载、关闭按钮

3. 连线精确对接
   - 连线现在精确连接到参数端口位置
   - 新增 toPortKey 支持端口级别连接
   - 文字节点连线自动填充内容

4. 已连接状态显示
   - 参数连线成功后显示“已连接”状态
   - 显示链接图标和内容预览
   - 锁定状态防止误操作

🎨 UI 改进
--------
1. 自定义下拉框
   - 替换原生 select 为深色主题下拉框
   - 与节点整体风格适配
   - 支持点击外部关闭

2. 输入控件优化
   - STRING 类型使用文本输入框
   - LIST/COMBO 类型使用下拉选择框
   - IMAGE/VIDEO/AUDIO 类型支持上传和连线

3. 连接点位置修正
   - 输入端口统一在节点左侧
   - 支持拖拽连线输入

🔧 修复
--------
1. 修复 fieldData JSON 数组格式解析问题
2. 修复连线渲染位置计算
3. 修复控制条被 overflow-hidden 裁剪的问题
4. 修复更新弹窗可拖动的问题
5. 优化更新弹窗滚动条样式和内容滚动

📝 技术改进
--------
1. Connection 类型扩展 toPortKey 字段
2. 优化端口 Y 坐标计算逻辑
3. 重构 rh-config 节点渲染结构

---
感谢使用企鹅工坊！
